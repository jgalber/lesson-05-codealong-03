ICONS

Icons are very useful to implement in our designs, specially to link to our social media links.

fontawesome 5 is the newest version and starts with 'fas, fab, far', then 'fa' 'dashed' and name of the icon

fontawesome 5 is added by 'script' tag in the head section, and the icon tag 'i' is accompanied by class attribute for styling

fontawesome 4 is the previous version and starts simply with 'fa', then 'fa' 'dashed' 'icon-name'


fontawesome 4 is added by 'link' tag, rel 'stylesheet' and followed by the icon web source, it resides within the head section as a link, in the body take a place the icon tag 'i' that is accompanied by a class attribute for styling

w3schools Icon reference

Weird performance of styling with w3schools icons, the only way to style the icon is 'inline' that doesn't allow to style internal or external css code.
